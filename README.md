## Project Overview
This project is a full stack web application built as part of the Full Stack Developer Capstone. It includes:
- A Django backend that exposes APIs for dealer and review data.
- A React frontend that consumes the backend APIs and provides user-facing pages such as Home, About, Contact, Sign-up, Login, and Reviews.
- Integration with external services for retrieving dealership information and performing sentiment analysis on customer reviews.
